# ISP2024 examples

This file gives minimal examples for building datasets from the 2024 Integrated System Plan (ISP) datafiles.

## By planning year
```julia
using ParseISP

reference_trace = 4006
poe = 10
target_years = [2030, 2031]

ParseISP.build_datasets(
    ParseISP.ISP2024(),
    downloadpath = joinpath(@__DIR__, "..", "data", "parseisp-downloads"),
    poe          = poe,
    reftrace     = reference_trace,
    years        = target_years,
    output_root  = joinpath(@__DIR__, "..", "data", "parseisp-datasets"),
    write_csv    = true,
    write_arrow  = false,
    scenarios    = [1, 2, 3],
)
```

## By date range
Use `drange` to build datasets for explicit date windows instead of full calendar years.

```julia
using ParseISP

ParseISP.build_datasets(
    ParseISP.ISP2024(),
    downloadpath = joinpath(@__DIR__, "..", "data", "parseisp-downloads"),
    poe          = 10,
    reftrace     = 4006,
    drange       = [("01-01-2030", "31-03-2030"), ("01-07-2031", "30-09-2031")],
    output_root  = joinpath(@__DIR__, "..", "data", "parseisp-datasets"),
    write_csv    = true,
    write_arrow  = false,
    scenarios    = [1, 2, 3],
)
```

`years` and `drange` are mutually exclusive, and one of them must be supplied.
